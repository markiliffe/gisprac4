Integrating and Visualising Bus Mapping Data

This page demos the the various maps created as part of the GI Services course and coursework. It uses maps from various sources, like MapBox, TileMill and others. Some data used is UK Government Open Data (LSOA Boundaries), some from the University of Nottingham (Deprevation Indices) and MapData, where used, is (c) ODBl OpenStreetMap contributors.
